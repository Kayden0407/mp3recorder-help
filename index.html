F.A.Q.

How to use this software properly?

This software is made specially to record audio for oral. Follow the instructions below on how to use the software.

    • Click on the big button to start recording

        *Note: You will need to allow access for the site to use your microphone (there will be a prompt)

      Say what you need to say
      Click the red button to stop recording once you are done
      Click on the downward pointing blue button to download your audio
    • How to I get to all the apps of this package (PCPS)?

    The package name is PCPS. To get to all the apps and the dashboard, click on the button that says 'PCPS'.
